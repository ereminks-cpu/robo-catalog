## Что сделать

1. В Supabase SQL Editor один раз выполнить:
   `supabase_admin_tools_setup.sql`

2. На GitHub заменить:
   - `index.html`
   - `style.css`
   - `README.md`

3. После публикации сделать жесткое обновление:
   `Cmd + Shift + R`
