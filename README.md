# ROBO Catalog v9.11 — Admin Tools

Добавлено для Admin:

1. Удаление робота из базы
- кнопка `Удалить робота` в карточке;
- удаляет запись из `public.robots`;
- также пытается удалить главное фото, галерею и файлы из Storage.

2. Справочник производителей и стран
- отдельная вкладка в админке;
- добавление / изменение / удаление производителя;
- страна автоматически подставляется в карточке по выбранному производителю;
- при изменении страны производителю обновляются карточки этого производителя.

## Перед загрузкой сайта

В Supabase SQL Editor выполнить:

`supabase_admin_tools_setup.sql`
