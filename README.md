# ROBO Catalog v10.40.1 — Solution Advisor UI

Первый этап модуля «Подбор решения».

## Что добавлено

- Новая вкладка `Подбор решения`.
- Отдельный интерфейс мастера подбора.
- 4 шага:
  1. Объект
  2. Параметры
  3. Инфраструктура
  4. Экспертное заключение
- Демонстрационный экран ROBO Recommendation.

## Важно

В этой версии расчет еще не подключен. Это UI-only релиз для согласования интерфейса.

## Что НЕ изменялось

- каталог;
- таблица каталога;
- карточка робота;
- дашборд;
- сравнение;
- администрирование;
- Supabase-структура.

SQL не требуется.
