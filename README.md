# ROBO Catalog v10.50 — Solution Engine v1

Первый рабочий движок подбора решения.

## Что добавлено

- Чтение профиля `Подземный паркинг` из Supabase.
- Чтение критериев из `solution_criteria`.
- Проверка всех роботов каталога.
- Обязательные критерии типа `Фильтр` исключают неподходящих роботов.
- Балльные критерии считают процент соответствия.
- Вместо демо-карточек выводится реальный ТОП роботов.
- В результатах показывается:
  - выполнено;
  - не выполнено;
  - нет данных.
- Сводка:
  - проверено роботов;
  - после фильтра;
  - исключено;
  - количество критериев.

## SQL

Новый SQL не требуется, если уже выполнен SQL v10.30.

## Что НЕ изменялось

- каталог;
- карточка робота;
- дашборд;
- администрирование;
- Supabase-структура.
