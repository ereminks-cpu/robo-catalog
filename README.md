## Что добавлено

- Профили подбора в Supabase.
- Реляционная таблица критериев `solution_criteria`.
- Создание, переименование, копирование и удаление профилей.
- Добавление критериев только из полей карточки робота.
- Удаление любого критерия из профиля.
- Изменение типа: фильтр / балльный.
- Изменение веса, условия и порога.
- Контроль суммы весов.
- Проверка профиля.
- Сортировка критериев А→Я / Я→А с сохранением порядка через `sort_order`.
