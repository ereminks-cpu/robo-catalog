# ROBO Catalog v10.20 — Criteria Admin

Добавлено: администрирование критериев подбора.

## Что сделано

В существующем разделе `Администрирование` добавлена третья вкладка:

- Пользователи
- Производители и страны
- Критерии подбора

## Возможности

- создавать профили подбора;
- удалять профили;
- предустановленный профиль `Подземный паркинг`;
- удалять любой критерий внутри профиля;
- добавлять критерий только из базы полей карточки робота;
- менять:
  - активность критерия;
  - тип: фильтр / балльный;
  - вес;
  - условие;
  - порог.

## Важно

Для облачного хранения критериев нужно один раз выполнить в Supabase SQL Editor:

`supabase_solution_profiles.sql`

Если SQL не выполнен, вкладка будет работать во временном режиме через браузерный localStorage.
