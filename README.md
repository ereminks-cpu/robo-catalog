# ROBO Catalog v10.32 — Criteria Admin Fixes

Исправления администрирования критериев.

## Что исправлено

- Убран дубль критерия `Проблесковый маячок`.
- Если в старых данных встречается `beaconLight`, он автоматически приводится к `warningBeacon`.
- Для справочных полей теперь доступен множественный выбор значений:
  - Тип эксплуатации
  - Назначение
  - Подтип
  - Статус
  - Производитель
  - Страна
  - Тип навигации
  - Тип моющего узла
  - FMS / лифт / работа в толпе и др.
- Значения берутся из справочников карточки и из фактических данных роботов.
- При добавлении нового критерия заполненные параметры предыдущих критериев не сбрасываются.
- Вес критериев обновляется в реальном времени при изменении веса, типа или активности критерия.
- После сохранения вес также обновляется.

## SQL

Новый SQL не требуется, если уже выполнен SQL v10.30.

## Что НЕ изменялось

- основной дизайн каталога v09.25;
- таблица каталога;
- карточка робота;
- сравнение;
- дашборд;
- медиа и загрузка файлов.
