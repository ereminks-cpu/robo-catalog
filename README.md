# ROBO Catalog v10.31 — Criteria Admin UI Fix

Исправления интерфейса администрирования критериев.

## Что исправлено

- В выпадающем списке критериев убраны англоязычные технические названия.
- Поля сгруппированы по разделам:
  - Паспорт
  - Характеристики
  - Функции уборки
  - Навигация
  - Безопасность
  - Станция
  - Алгоритмы
  - ROBO-Аналитика
- Внутренние имена полей показываются второстепенно в таблице, а не как основное название.
- Тип данных теперь отображается по-русски: Да/Нет, Число, 0–3 балла и т.д.
- При добавлении нового критерия ранее заполненные параметры больше не сбрасываются.
- Если выбран тип `Фильтр`, поле `Вес` автоматически становится неактивным и принимает значение `0`.
- Для boolean-полей порог отображается как `Да/Нет`, а не `true/false`.
- Плашка суммы весов сделана более понятной.

## SQL

Новый SQL не требуется, если уже выполнен SQL v10.30.
