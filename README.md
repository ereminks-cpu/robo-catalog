# ROBO Catalog с ручной загрузкой изображений

Это статический сайт для GitHub Pages.

## Что изменилось
- Добавлена колонка «Фото».
- Пользователь может нажать «Добавить фото» в строке модели.
- Фото сохраняется в браузере пользователя через LocalStorage.
- Можно экспортировать базу фото в JSON и импортировать ее на другом компьютере.

## Важно
На GitHub Pages без backend/CMS фото не сохраняются централизованно для всех пользователей.
Чтобы фото стали общими для всех, нужно либо:
1. загрузить изображения в папку `images/` и прописать их в коде;
2. подключить backend: Supabase/Firebase/Netlify CMS/GitHub API.
