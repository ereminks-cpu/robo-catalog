# ROBO Catalog v10.33 — Multi-select Criteria Fix

Исправление выбора значений для справочных критериев.

## Что исправлено

- Стандартный неудобный `select multiple` заменен на понятный чекбокс-список.
- Для справочных критериев появились:
  - Выбрать все
  - Снять все
  - счетчик выбранных значений
- Для справочных критериев оператор автоматически фиксируется как `Любое`.
- Администратору больше не нужно использовать Ctrl/Cmd для множественного выбора.

## SQL

Новый SQL не требуется, если уже выполнен SQL v10.30.
